# Website Cycling Club Waregem

The official website of Cycling Club Waregem, a recreational cycling team in the city of Waregen, Belgium.

This single page website was a side project and uses just basic HTMl in combination with TailwindCSS.
[WIP] You can reach the website [here](https://cyclingclubwaregem.be)
